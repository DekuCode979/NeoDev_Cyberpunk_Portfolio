# 🚀 Cyberpunk Portfolio

## ✨ Introducción
El **Portafolio Cyberpunk** es una página web interactiva que combina diseño futurista con funcionalidad técnica. Desarrollado con **HTML, CSS y JavaScript**, busca mostrar habilidades, proyectos y contacto de manera clara y atractiva, utilizando una estética oscura con acentos neón y animaciones dinámicas.

## 🎯 Objetivos principales
- Interactividad: Animaciones de scroll reveal, barras de progreso dinámicas y modales personalizados.  
- Estética Cyberpunk: Paleta oscura con acentos neón (#00ffff, #ff00ff).  
- Presentación Profesional: Secciones claras: Hero, About, Skills, Projects y Contact.  
- Experiencia Técnica: Efectos de fade-in, slide-up y sonido retro en modales.  

## 🛠️ Desarrollo
1. **Diseño visual:** Paleta oscura con tonos neón para transmitir energía y modernidad.  
2. **Estructura HTML:** Secciones organizadas: Hero, About, Skills, Projects y Contact.  
3. **Estilos CSS:** Barras de progreso con formato `0/5`, animaciones de fade-in y slide-up, hover effects con brillo neón.  
4. **Funcionalidad JS:** Scroll reveal, animación de barras de Skills, modales con sonido retro beep y animación slide-up.  

## 📂 Código fuente
- `index.html` → estructura del portafolio.  
- `style.css` → estilos cyberpunk y animaciones.  
- `script.js` → lógica de interactividad y modales.  

## ✅ Conclusión
El desarrollo del **Portafolio Cyberpunk** combina estética futurista con funcionalidad técnica.  
El mayor reto fue adaptar las imágenes de proyectos sin deformarse, resuelto con `object-fit: cover`.  
Este proyecto fortalece mi portafolio personal al demostrar dominio de **HTML, CSS y JavaScript**, junto con creatividad en la presentación visual.
